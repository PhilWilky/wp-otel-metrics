OpenTelemetry tracing for WordPress. Monitor performance, track errors, and gain insights into your WordPress application.

== Description ==

WP OpenTelemetry brings enterprise-grade observability to WordPress. Start with our free version and upgrade to Pro when you need advanced features.

= Free Features =
* Basic request tracing
* Jaeger/Zipkin integration
* Simple configuration
* Basic performance metrics

= Pro Features =
* Multiple backend support (DataDog, New Relic, etc.)
* Adaptive sampling
* Request batching
* Data compression
* Advanced performance monitoring
* Priority support

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wp-otel-metrics`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your tracing endpoint in Settings > OTel Metrics
4. (Optional) Enter your Pro license key to unlock advanced features
